# item-analysis-and-mmle-irt
Repository of item analysis and MMLE-IRT for my own V-ACT practice tests

Chú thích:
- Dữ liệu: 
    + Thông tin ghi nhận làm bài thi ở hai đề (lan1.csv và lan2.csv) và đáp án hai đề này (ans_1.csv và ans_2.csv)
    + File khảo sát các thí sinh sau khi tham gia thi thử (KhaoSat_Cleaned.xlsx)
- Mã nguồn Python: 1 file ctt.py thực hiện tính toán theo lý thuyết CTT, 1 file irt.py thực hiện chạy thực nghiệm mô hình IRT 2 tham số, 1 file item_plot.py vẽ biểu đồ.
- Notebook: 1 file main.ipynb thực hiện phân tích kết quả
