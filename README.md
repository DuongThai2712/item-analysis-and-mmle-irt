# item-analysis-and-mmle-irt
Repository of item analysis and MMLE-IRT for my own V-ACT practice tests

* **Data:**

  * Records of students’ responses for three test forms (lan1.csv, lan2.csv, and lan3.csv) and the corresponding answer keys (ans_1.csv, ans_2.csv, and ans_3.csv)
  * Post-mock exam survey file (KhaoSat_Cleaned.xlsx)

* **Python source code:**

  * `ctt.py`: performs calculations based on Classical Test Theory (CTT)
  * `irt.py`: runs experiments with the 2-parameter IRT model
  * `item_plot.py`: generates item analysis plots

* **Notebook:**

  * `main.ipynb`: conducts the result analysis

**Notes:** We cannot provide students' responses and survey file since they contain real information.
